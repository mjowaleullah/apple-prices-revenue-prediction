# Apple Prices Revenue Prediction

This project focuses on building a Machine Learning model to predict **Revenue** using the Apple Prices Dataset.

## 📋 Project Overview

The main objective of this project is to predict the **Revenue** (total income) generated from apple sales based on various features such as price per unit, quantity sold, market type, region, and other factors.

The dataset used is **synthetic** (artificially generated) and contains information about apple pricing and sales across different markets and countries.

## 📊 Dataset

- **Dataset Name**: Apple Prices Dataset
- **Number of Records**: 500+
- **Time Period**: 2024–2025
- **Features**:
  - Country / Region
  - Market Type
  - Apple Variety
  - Price per unit
  - Quantity Sold
  - Date
  - Others (Season, Demand Level, etc.)
- **Target Variable**: `Revenue` = Price × Quantity

## 🛠️ Technologies & Libraries Used

- Python 3
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn (for EDA)
- Jupyter Notebook
- (Optional) XGBoost, LightGBM, Random Forest, Linear Regression, LSTM, etc.
